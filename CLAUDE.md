An interactive, static reference for Argentine tango, built with Astro and deployed to GitHub Pages. The core interface is a navigable graph of dance figures: nodes ("blobs") are moves, edges show how moves connect, and clicking a node opens a detail panel with a description, a short silent looping video, and a numbered step diagram. The source of truth is structured data — language-neutral YAML files describing figures (each with entry/exit "state" preconditions drawn from a controlled vocabulary) plus per-language Markdown content trees — from which everything is generated, including a computed compatibility graph and a build-time validator. The same content renders in two modes: an animated, expressive version, and a calm high-contrast low-motion version that respects prefers-reduced-motion; and in multiple languages. The project is edited continuously, so the workflow must be "edit a data file, commit, push, it's live."
